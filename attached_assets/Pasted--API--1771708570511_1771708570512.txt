### نکته‌ها
- **Poyo AI**: بیشتر برای ویدیو/تصویر است، نه متن. اگر نیاز به مدل متن دارید، از OpenRouter استفاده کنید.
- **OpenRouter**: بهترین گزینه برای مدل‌های رایگان متن/چت. لیست بالا از جستجوهای به‌روزرسانی‌شده است (فوریه ۲۰۲۶).
- **Portkey**: gateway است، نه ارائه‌دهنده مدل. free tier برای پلتفرم (لاگین، مدیریت) است، اما مصرف مدل‌ها هزینه دارد مگر open-source محلی باشد.

### تغییرات کد `testProvider` (برای جلوگیری از خطاها)
- چک کلید قبل از fetch اضافه شد.
- مدل‌های نامعتبر رو هندل می‌کنه.
- پرامپت تست رو ساده‌تر کردم.

```typescript
const testProvider = async (id: string, retryWithBackup = true) => {
  // ... کد قبلی ...
  try {
    // چک کلید
    if (!apiKey || apiKey.trim() === '') {
      throw new Error(`No valid API key for ${id}`);
    }

    // چک مدل موجود
    if (!provider.models.includes(model)) {
      throw new Error(`Model ${model} not available for ${id}`);
    }

    // fetch ...
  } catch (error: any) {
    // ... retry ...
  }
};
```

حالا داشبورد رو با این تغییرات اجرا کن – فقط مدل‌های رایگان رو تست می‌کنه. اگر باز خطا داشت، لاگ دقیق رو بفرست. 🚀
